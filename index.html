About

The OpenBusiness Framework aims to help developers and businesses by providing a toolset for rapid application generation.

Each component has the following sub components which can be used separately:

  • Visual designer
  • Configuration parser
  • Code generator

Components

Modeling engine

The modeling engine enables users to easily create and edit domain models. Several returing business concepts are already made available.

Business process engine

The business process engine handles business processes and related tasks. Processes can be created by visually designing them.

Process designer

Portal generation engine

The portal generation engine generates (web)portals.

Service generation engine

The service generation engine generates RESTFul- or web-services which can be used separately or through one of the other components.

Core concepts

Code generation core

The code generation core generates code based on a given configuration (which can be based on a visual design model).
